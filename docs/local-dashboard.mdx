---
title: Local Dashboard
description: Review runs in a local-only web dashboard.
---

The dashboard is served by the local daemon and is **local-only**. It binds to
`127.0.0.1` (never `0.0.0.0`), prefers port `8757`, and falls back to the next
available port (`8758`, `8759`, …) if that is busy. There is no account and no
authentication because it is reachable only from your machine.

Open it with:

```bash
trace dashboard
```

## Development mode

You can run the dashboard against a running daemon with Vite's dev server:

```bash
cd apps/web
npm install
npm run dev      # http://localhost:5173, proxies /api to the daemon
```

In production the daemon serves the built static files from `apps/web/dist`.
