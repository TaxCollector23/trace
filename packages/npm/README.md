# trace (npm wrapper)

Optional npm install path for the Trace CLI. Installs the platform `trace`
binary from GitHub Releases.

```bash
npm install -g trace
trace --help
```

This is a **fallback** install method. The primary methods are Homebrew (macOS),
the PowerShell script (Windows), and the curl shell script (Linux/macOS). See
the [main README](../../README.md).

Pin a version with `TRACE_VERSION=v1.1.0 npm install -g trace`.
